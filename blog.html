3. What is event delegation in the context of the DOM, and why is it useful?

Answer: Event delegation means at which element addEventListener is being added. Sometimes, adding event to the parent node give us some benefits. In that case, event is to be added to the parent node instead child nodes. For example, while adding child nodes to parent nodes, event is to be delegated to the parent node. It is useful because It will save some unusual codings and results in better performance. Read More...
